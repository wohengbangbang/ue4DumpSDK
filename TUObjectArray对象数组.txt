class UObject{
	PVOID VTable;			// 虚表指针 +0x0       
	UClass *ClassPrivate; 		// 当前对象的类型 +0x10
	int32_t InternalIndex;   	// 对象在GUObjectArray中的序号
	FName NamePrivate;		//对象的名字 +0x18
	...
	UClass *ClassPrivate;		// 对象的父类 +0x40
} 



class UField : public UObject{
	UField *Next;		// 下一成员的指针 +0x20
}









UObject	内存中实际存在的对象	一辆真实的汽车
UClass	描述对象有哪些属性、方法的"图纸"	汽车的设计图纸
FField	图纸上的一条描述（属性、函数等）	图纸上的一个规格条目
FProperty	专门描述"属性"的FField	图纸上的"颜色"规格
FFieldClass	描述这个属性是什么类型（int、string等）	"颜色"这个规格是"字符串类型"
